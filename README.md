# Beginner Java Counter App (Android)

A fundamental, beginner-friendly mobile application built using native Java for Android. This project is intentionally kept simple to isolate and demonstrate the most core principles of mobile development.

# Key Learning Concepts Explored
View Binding: Connecting UI visual components designed in XML (`findViewById`) directly to actionable Java controller variables.
Event Listeners: Implementing an `OnClickListener` interface to intercept and respond to human-to-device physical touchscreen interactions.
State Management: Tracking runtime state changes (an evolving integer data type) and instantly feeding those updates back into the user interface view layer dynamically.
